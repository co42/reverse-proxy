Simple reverse proxy in Rust with actix-web

```
Simple reserve proxy

USAGE:
    reverse-proxy [OPTIONS]

OPTIONS:
    -a, --address <ADDRESS>    Bind address [default: 127.0.0.1]
    -h, --help                 Print help information
    -p, --port <PORT>          Listen on port [default: 4242]
    -t, --to <TO>              Proxy requests to [default: http://localhost:8000]
```
